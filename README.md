
## OpenWrt for HG680-P & B860H

|Firmware information
|----------------------------
|Default IP : 192.168.1.1
|Default Username : root
|Default Password : passwd
|WiFi SSID : LYNX
|WiFi Password : none
|Install to EMMC : Login to OpenWrt → System → Amlogic Service → Install OpenWrt
